# Система управления личными тренировками (Workout Tracker)

Веб-приложение для учёта и планирования тренировок, разработанное на Flask.

## Описание проекта

Приложение позволяет пользователям:
- Вести учёт своих тренировок
- Добавлять упражнения с характеристиками
- Создавать тренировочные программы
- Отслеживать прогресс
- Получать рекомендации по нагрузке на основе личных параметров
- Экспортировать отчёты о тренировках

## Технологический стек

- **Backend**: Flask 3.0
- **Database**: MySQL 8.0
- **Frontend**: HTML5, CSS3, JavaScript
- **Templates**: Jinja2

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone <repository-url>
cd WorkoutTracker
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка базы данных

Создайте базу данных MySQL и выполните SQL-скрипт:

```bash
mysql -u root -p < database.sql
```

### 5. Настройка окружения

Скопируйте `.env.example` в `.env` и заполните параметры:

```bash
cp .env.example .env
```

### 6. Запуск приложения

```bash
python app.py
```

Приложение будет доступно по адресу: http://localhost:5000

## Функциональность

### Для пользователей:
- Регистрация и авторизация
- Создание собственных упражнений
- Планирование тренировок
- Калькулятор оптимальной нагрузки
- Отчёты по тренировкам
- Экспорт данных в CSV

### Для тренеров:
- Все возможности пользователя
- Создание публичных упражнений
- Модерация контента
- Управление базой упражнений
