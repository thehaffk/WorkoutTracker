Форма регистрации нового пользователя в системе

Для создания новой учётной записи в информационной системе управления тренировочным процессом необходимо заполнить все обязательные поля регистрационной формы, расположенной ниже, в соответствии с установленными требованиями к формату вводимых данных. После успешного завершения процедуры регистрации будет осуществлён автоматический переход к форме авторизации для последующего входа в систему с использованием созданных учётных данных.

Имя пользователя должно содержать от 3 до 50 символов и может включать только латинские буквы (как заглавные, так и строчные), цифры, символы подчёркивания (_) и дефиса (-). Данное имя будет использоваться для входа в систему.
Укажите действующий адрес электронной почты, который будет использоваться для идентификации учётной записи и восстановления доступа к системе в случае необходимости.
Пароль должен содержать не менее 8 символов и обязательно включать в себя как минимум одну заглавную букву латинского алфавита, одну строчную букву латинского алфавита и одну цифру. Рекомендуется использовать сложные пароли для обеспечения надлежащего уровня безопасности учётной записи.
В целях предотвращения ошибок при вводе пароля необходимо повторно ввести тот же самый пароль, который был указан в предыдущем поле. Значения в обоих полях должны полностью совпадать.
При установке данного флажка учётной записи будут предоставлены дополнительные права и функциональные возможности, предназначенные для пользователей с ролью "Тренер", включая доступ к расширенным инструментам управления тренировочным процессом.

В случае если учётная запись в системе уже была создана ранее, можно осуществить переход к форме авторизации.
