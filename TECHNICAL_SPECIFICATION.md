## 1. АНАЛИЗ ПРЕДМЕТНОЙ ОБЛАСТИ

### 1.1 Обзор существующих программных продуктов

В настоящее время существует множество приложений для отслеживания тренировок:

**MyFitnessPal**
- Бесплатно: базовый учёт активности, калории
- Платно: персональные планы, расширенная аналитика, без рекламы

**JEFIT**
- Бесплатно: база упражнений, планирование тренировок
- Платно: расширенная статистика, синхронизация между устройствами

**Strong**
- Бесплатно: базовый учёт тренировок
- Платно: графики прогресса, экспорт данных, резервные копии

**FitNotes**
- Полностью бесплатное приложение
- Офлайн работа, но ограниченная визуализация

**Недостатки существующих решений:**
- Большинство функций за платной подпиской
- Сложный интерфейс для начинающих
- Отсутствие гибкости в создании упражнений
- Ограниченные возможности экспорта данных

## 2. ПРОЕКТИРОВАНИЕ

### 2.1 Целевая аудитория

**Характеристики:**
- Возраст: 18-50 лет
- Интересы: фитнес, спорт, здоровье
- Цели: отслеживание прогресса, планирование тренировок
- Уровень: от новичков до продвинутых спортсменов

**Группы пользователей:**
1. **Начинающие спортсмены** - нужна простота и рекомендации
2. **Опытные атлеты** - детальная статистика и гибкость
3. **Тренеры** - управление базой упражнений, помощь клиентам
4. **Люди на диете** - отслеживание калорий

### 2.2 User Stories

**Обычный пользователь:**
- Хочу зарегистрироваться для доступа к системе
- Хочу добавлять свои упражнения с параметрами
- Хочу создавать тренировки и добавлять в них упражнения
- Хочу видеть свой прогресс по тренировкам
- Хочу использовать калькулятор для определения оптимальной нагрузки
- Хочу экспортировать статистику за период

**Тренер:**
- Хочу создавать публичные упражнения для всех пользователей
- Хочу модерировать базу упражнений
- Хочу видеть все возможности обычного пользователя

### 2.3 Модель данных

**Сущности:**
1. **users** - пользователи системы
2. **exercises** - упражнения (публичные и личные)
3. **workouts** - тренировочные сессии
4. **m2m_exercises_workouts** - связь упражнений и тренировок

**Связи:**
- users → exercises (1:M) - пользователь создаёт упражнения
- users → workouts (1:M) - пользователь создаёт тренировки
- exercises ↔ workouts (M:M) - упражнения в тренировках

### 2.4 Wireframe макеты

Созданы макеты для следующих страниц:
- Вход и регистрация
- Список упражнений
- Создание/редактирование упражнения
- Список тренировок
- Создание/редактирование тренировки
- Калькулятор нагрузки
- Отчёты

## 3. ФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ

### 3.1 Аутентификация
- Регистрация с валидацией данных
- Вход по username или email
- Хеширование паролей
- Сессии пользователей

### 3.2 CRUD для упражнений
- Создание упражнений с полями:
  - Название
  - Группа мышц
  - Оборудование
  - Сложность
  - Калории за подход
  - Описание техники
  - Изображение
- Просмотр списка упражнений
- Редактирование упражнений
- Удаление упражнений
- Публичные/личные упражнения

### 3.3 Управление тренировками
- Создание тренировки с параметрами:
  - Дата
  - Тип тренировки
  - Длительность
  - Интенсивность
- Добавление упражнений в тренировку
- Указание подходов, повторений, веса
- Автоматический расчёт общей нагрузки
- Просмотр деталей тренировки
- Редактирование и удаление

### 3.4 Калькулятор нагрузки
- Ввод параметров: возраст, вес, рост, пол
- Выбор уровня опыта
- Выбор цели (похудение/поддержание/набор/сила)
- Расчёт рекомендаций:
  - Количество тренировок в неделю
  - Длительность тренировки
  - Интенсивность
- Сохранение рекомендаций в профиле

### 3.5 Отчёты и статистика
- Выбор периода (дата начала и конца)
- Просмотр таблицы тренировок за период
- Суммарная статистика:
  - Количество тренировок
  - Общее время
  - Сожжённые калории
  - Поднятый вес
- Экспорт в CSV

### 3.6 Роль тренера
- Создание публичных упражнений
- Модерация пользовательских упражнений
- Доступ ко всем упражнениям системы

## ОТЛИЧИЯ ОТ ДНЕВНИКА ПИТАНИЯ

Данная тема отличается от темы "Дневник питания" следующим:

1. **Предметная область**: Тренировки vs Питание
2. **Основная сущность**: Упражнения vs Продукты
3. **Параметры**: Подходы/повторения/вес vs Калории/БЖУ/граммы
4. **Калькулятор**: Оптимальная нагрузка vs БМО
5. **Цели**: Сила/выносливость vs Похудение/набор веса
6. **Метрики**: Поднятый вес, время тренировки vs КБЖУ
7. **Роли**: Тренер vs Диетолог/Администратор
8. **Отчёты**: Статистика тренировок vs Статистика питания
